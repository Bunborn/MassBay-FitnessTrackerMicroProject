Fitness Tracker

Analyze fitness data using MATLABÂ® functions

Load and analyze accelerometer and GPS data from MATLAB Mobile. Compute step counts, distances, and classify activities using machine learning.

Architecture Overview

The Fitness Tracker is organized into four MATLAB packages, each with a specific responsibility:

Package Purpose Contents
+data Data Loading & Cleaning Functions to load raw sensor data from .mat files, validate structure, and prepare timetables for analysis
+analysis Analysis Functions Classes that implement the AnalysisFunction interface for processing fitness data (step counting, distance calculation, activity classification)
+train Offline Model Training Scripts to train machine learning models from labeled data; models are saved and used by analysis classes at runtime
+utils Helper Functions Utility functions shared across packages (e.g., time conversions)

Data Flow

% 1. Load raw data
fitnessData = data.loadFitnessData('ExampleData.mat');

% 2. Create an analysis object
analyzer = analysis.StepCounter();

% 3. Run analysis
results = analyzer.analyze(fitnessData);

% 4. Visualize results
analyzer.plot();

AnalysisFunction Interface

All analysis classes inherit from AnalysisFunction and implement a common interface:

MethodDescription
analyze(fitnessData)Process data and return results struct
plot()Generate default visualization
displayResults()Print summary to command window
getResults()Retrieve results struct

This consistent interface allows you to swap analysis classes without changing your workflow.

Quick Start

% Setup paths
setupPaths();

% Load fitness data
fitnessData = data.loadFitnessData('data/ExampleData.mat');

% Count steps
stepCounter = analysis.StepCounter();
results = stepCounter.analyze(fitnessData);
stepCounter.plotSteps();

Data Loading (+data)

  • loadFitnessData
    Load and validate fitness data from .mat files

Syntax

fitnessData = data.loadFitnessData(dataPath)
fitnessData = data.loadFitnessData(dataPath, 'ValidateData', false)

Output Fields

FieldDescription
AccelerationTimetable with X, Y, Z acceleration
PositionTimetable with latitude, longitude
hasAccelerationTrue if acceleration data present
hasPositionTrue if GPS data present

Analysis Classes (+analysis)

All classes inherit from AnalysisFunction with methods: analyze(), plot(), displayResults()

  • AccelerationAnalysis
    Compute acceleration magnitude and statistics
  • StepCounter
    Detect steps using peak detection on acceleration
  • GPSDistanceCalculator
    Calculate distance from GPS coordinates
  • ActivityClassifier
    ML-based activity classification (sitting, walking, running)

StepCounter Properties

PropertyDefaultDescription
Threshold1.2Minimum peak height for step detection

PropertyDefaultDescription
