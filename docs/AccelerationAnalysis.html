AccelerationAnalysis

Compute acceleration magnitude and statistics

Analyzes 3-axis accelerometer data to compute magnitude and statistical measures. Inherits from AnalysisFunction.

Syntax

analyzer = analysis.AccelerationAnalysis()
results = analyzer.analyze(fitnessData)
analyzer.plotMagnitude()
analyzer.plotComponents()

Description

AccelerationAnalysis computes the acceleration magnitude from X, Y, Z components using √(X² + Y² + Z²) and provides statistical analysis including mean, standard deviation, min, max, and median values.

Methods

MethodDescription
analyze(fitnessData)Run analysis on fitness data, returns results struct
plotMagnitude()Plot acceleration magnitude over time
plotComponents()Plot X, Y, Z components separately
plot()Default plot (calls plotMagnitude)
displayResults()Print results to console
getResults()Return results structure

Output Arguments

FieldTypeDescription
magnitudedouble arrayComputed acceleration magnitude √(X² + Y² + Z²)
Xdouble arrayX-axis acceleration data
Ydouble arrayY-axis acceleration data
Zdouble arrayZ-axis acceleration data
timeDatadatetime arrayTimestamps for each sample
statsstructStatistics: mean, std, max, min, median

Examples

Basic Usage

% Load data and analyze
fitnessData = data.loadFitnessData('data/ExampleData.mat');

analyzer = analysis.AccelerationAnalysis();
results = analyzer.analyze(fitnessData);

% Display statistics
fprintf('Mean acceleration: %.2f g\n', results.stats.mean);
fprintf('Max acceleration: %.2f g\n', results.stats.max);

Visualization

% Plot magnitude and components
figure;
subplot(2,1,1);
analyzer.plotMagnitude();

subplot(2,1,2);
analyzer.plotComponents();
