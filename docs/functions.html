Data Loading (+data)

Functions for loading and preparing fitness data.

Function Description
data.loadFitnessData Load and validate fitness data from .mat files. Returns a struct with Acceleration and Position timetables.

loadFitnessData

Syntax:

fitnessData = data.loadFitnessData(dataPath)
fitnessData = data.loadFitnessData(dataPath, 'ValidateData', false)

Input Arguments:

ArgumentTypeDescription
dataPathstringPath to .mat file containing sensor data
ValidateDatalogical(Optional) Validate and structure data. Default: true

Output:

FieldTypeDescription
AccelerationtimetableX, Y, Z acceleration data with timestamps
PositiontimetableLatitude, longitude data with timestamps
hasAccelerationlogicalTrue if acceleration data present
hasPositionlogicalTrue if GPS data present

Analysis Classes (+analysis)

Classes for analyzing fitness data. All inherit from AnalysisFunction.

Class Description
analysis.AccelerationAnalysis Compute acceleration magnitude and statistics from 3-axis accelerometer data
analysis.StepCounter Detect and count steps using peak detection on acceleration magnitude
analysis.GPSDistanceCalculator Calculate distance traveled from GPS latitude/longitude coordinates
analysis.ActivityClassifier ML-based classification of activities (sitting, walking, running)

Model Training (+train)

Scripts for training machine learning models offline.

Function Description
train.trainActivityModel Train and save the activity classification model

trainActivityModel

Syntax:

train.trainActivityModel

Description: Trains a Decision Tree classifier on labeled activity data and saves the model to +analysis/models/activityModel.mat. This only needs to be run once; the saved model is used by ActivityClassifier at runtime.
