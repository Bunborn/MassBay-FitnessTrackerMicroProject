ActivityClassifier

ML-based activity classification

Classifies acceleration data into activity types (sitting, walking, running) using a trained machine learning model. Requires a pre-trained model file. Inherits from AnalysisFunction.

Note: This class requires a trained model. Run train.trainActivityModel first to generate the model file.

Syntax

classifier = analysis.ActivityClassifier()
results = classifier.analyze(fitnessData)
classifier.plotDistribution()
classifier.plotTimeline()

Methods

MethodDescription
analyze(fitnessData)Classify activities and return results struct
plotDistribution()Pie chart of activity distribution
plotTimeline()Activity over time visualization
plot()Default plot (calls plotDistribution)
displayResults()Print activity breakdown to console
getResults()Return results structure

Output Arguments

FieldTypeDescription
predictionscell arrayPredicted activity for each sample
uniqueActivitiescell arrayList of detected activity types
activityCountsdouble arrayCount of samples per activity
activityPercentagedouble arrayPercentage breakdown by activity

Activity Types

ActivityDescription
sittingLow acceleration, minimal movement
walkingModerate, rhythmic acceleration
runningHigh acceleration, faster rhythm

Examples

Basic Classification

% Load activity data
fitnessData = data.loadFitnessData('data/ActivityLogs.mat');

% Classify activities
classifier = analysis.ActivityClassifier();
results = classifier.analyze(fitnessData);

% Display breakdown
classifier.displayResults();

Visualization

% Create activity visualizations
classifier = analysis.ActivityClassifier();
classifier.analyze(fitnessData);

figure('Position', [100 100 1000 400]);

subplot(1,2,1);
classifier.plotDistribution();

subplot(1,2,2);
classifier.plotTimeline();

Training the Model

% If model doesn't exist, train it first
train.trainActivityModel

% Then use the classifier
classifier = analysis.ActivityClassifier();

Model Details

The classifier uses a Decision Tree model trained on labeled acceleration data. Features used for classification are X, Y, and Z acceleration values. The model file is stored at analysis/models/activityModel.mat.
