Examples

Learn how to use Fitness Tracker through practical examples

These examples demonstrate common workflows for analyzing fitness data. Each example includes complete, runnable code.

Getting Started

Set up your environment and load sample data.

% Add Fitness Tracker to path
setupPaths();

% Load the example dataset
fitnessData = data.loadFitnessData('data/ExampleData.mat');

% Inspect the loaded data
disp(fitnessData)

% Check what sensors are available
if fitnessData.hasAcceleration
    disp('Accelerometer data available');
    disp(fitnessData.Acceleration(1:5, :));  % Preview first 5 rows
end

if fitnessData.hasPosition
    disp('GPS data available');
    disp(fitnessData.Position(1:5, :));
end

Step Counting

Detect and count steps from accelerometer data using peak detection.

Basic Step Count

% Create step counter with default settings
stepCounter = analysis.StepCounter();

% Analyze the data
results = stepCounter.analyze(fitnessData);

% Display results
fprintf('Total steps: %d\n', results.stepCount);

% Visualize detected steps
figure;
stepCounter.plotSteps();
title('Detected Steps');

Adjusting Sensitivity

% Compare different threshold settings
thresholds = [1.0, 1.2, 1.5];
stepCounts = zeros(size(thresholds));

stepCounter = analysis.StepCounter();

for i = 1:length(thresholds)
    stepCounter.Threshold = thresholds(i);
    results = stepCounter.analyze(fitnessData);
    stepCounts(i) = results.stepCount;
end

% Display comparison
table(thresholds', stepCounts', 'VariableNames', {'Threshold', 'StepCount'})

GPS Distance Tracking

Calculate distance traveled from GPS coordinates.

Basic Distance Calculation

% Create GPS calculator
gpsCalc = analysis.GPSDistanceCalculator();

% Calculate distance
results = gpsCalc.analyze(fitnessData);

% Display results
fprintf('Distance: %.2f miles (%.0f feet)\n', ...
    results.totalDistanceMiles, results.totalDistanceFeet);
fprintf('Estimated steps: %d\n', results.estimatedSteps);

% Plot the route
figure;
gpsCalc.plotRoute();
title('GPS Route');

Customizing Stride Length

% Adjust stride length for different users
gpsCalc = analysis.GPSDistanceCalculator();

% Short stride (child or slow walk)
gpsCalc.StrideLength = 2.0;  % feet
results1 = gpsCalc.analyze(fitnessData);

% Long stride (tall adult or running)
gpsCalc.StrideLength = 3.0;
results2 = gpsCalc.analyze(fitnessData);

fprintf('Short stride estimate: %d steps\n', results1.estimatedSteps);
fprintf('Long stride estimate: %d steps\n', results2.estimatedSteps);

Activity Classification

Classify activities using machine learning.

Train the Model (One-Time Setup)

% Train the activity model (only needed once)
% This creates +analysis/models/activityModel.mat
train.trainActivityModel;

Classify Activities

% Create classifier (uses pre-trained model)
classifier = analysis.ActivityClassifier();

% Classify the data
results = classifier.analyze(fitnessData);

% Display activity breakdown
classifier.displayResults();

% Visualize activity distribution
figure;
subplot(1,2,1);
classifier.plotDistribution();

subplot(1,2,2);
classifier.plotTimeline();

Complete Analysis Workflow

Run all analyses on a dataset and create a summary dashboard.

%% Setup
setupPaths();
fitnessData = data.loadFitnessData('data/ExampleData.mat');

%% Run All Analyses
% Acceleration
accel = analysis.AccelerationAnalysis();
accelResults = accel.analyze(fitnessData);

% Step counting
steps = analysis.StepCounter();
stepResults = steps.analyze(fitnessData);

% GPS distance
gps = analysis.GPSDistanceCalculator();
gpsResults = gps.analyze(fitnessData);

% Activity classification
activity = analysis.ActivityClassifier();
activityResults = activity.analyze(fitnessData);

%% Create Dashboard
figure('Position', [100 100 1200 800], 'Name', 'Fitness Dashboard');

% Acceleration magnitude
subplot(2,2,1);
accel.plotMagnitude();
title('Acceleration Magnitude');

% Step detection
subplot(2,2,2);
steps.plotSteps();
title(sprintf('Steps Detected: %d', stepResults.stepCount));

% GPS route
subplot(2,2,3);
gps.plotRoute();
title(sprintf('Distance: %.2f miles', gpsResults.totalDistanceMiles));

% Activity distribution
subplot(2,2,4);
activity.plotDistribution();
title('Activity Breakdown');

%% Print Summary
fprintf('\n=== Fitness Summary ===\n');
fprintf('Mean acceleration: %.2f g\n', accelResults.stats.mean);
fprintf('Total steps: %d\n', stepResults.stepCount);
fprintf('Distance traveled: %.2f miles\n', gpsResults.totalDistanceMiles);
fprintf('Activities detected: %s\n', strjoin(activityResults.uniqueActivities, ', '));
