Examples

Learn how to use Fitness Tracker through practical demos

The Fitness Tracker includes two complete demo scripts that showcase all features. Choose the one that fits your workflow.

Interactive Dashboard

Launch the interactive dashboard for a modern UI experience with real-time analysis and visualization.

Quick Start

% Launch the dashboard app
launchDashboard

Features

  • Data Loading: Select and load .mat files from dropdown menu
  • Step Counter: Real-time step detection with adjustable sensitivity slider (5-40 m/sÂ²)
  • Activity Classification: Pie chart and timeline visualization of activities
  • GPS Route: Map visualization with total distance display
  • Acceleration Analysis: Multi-axis component plots with statistics

Usage

  1. Select a data file from the dropdown:
    • ExampleData.mat - Contains GPS and acceleration data
    • ActivityLogs.mat - Contains activity classification data
  2. Click "Load Data" to analyze and visualize
  3. Adjust the sensitivity slider to change step detection threshold
  4. Click "Refresh File List" to update available files

Note: The dashboard automatically detects which analyses can be performed based on available data (acceleration, GPS, orientation).

Command Line Demo

Run the comprehensive command-line demo that demonstrates all analysis functions with detailed visualizations.

Quick Start

% Run the complete demo script
fitnessTrackerDemo

What the Demo Shows

The fitnessTrackerDemo.m script demonstrates:

  • Data Loading: How to load and inspect fitness data
  • Acceleration Analysis: Magnitude and component visualization
  • Step Detection: Peak detection with customizable thresholds
  • GPS Distance Calculation: Route mapping and distance metrics
  • Activity Classification: Machine learning-based activity recognition
  • Summary Dashboard: Combined visualization of all analyses

Exploring the Code

Open fitnessTrackerDemo.m in the MATLAB editor to see complete, runnable examples of:

  • Creating and configuring analysis objects
  • Running analyses on fitness data
  • Accessing and displaying results
  • Creating custom visualizations
  • Comparing different analysis parameters

Customizing the Demo

You can modify the demo script to:

  • Use your own data files
  • Adjust analysis parameters (thresholds, stride length, etc.)
  • Add additional visualizations
  • Export results to files

Location

The demo script is located at: fitnessTrackerDemo.m in the project root directory.
